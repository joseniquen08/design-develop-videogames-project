## Nota sobre el audio
Los archivos de audio no están incluidos en este repositorio 
por derechos de autor. Para ejecutar el juego localmente, 
agrega tus propios archivos de audio en la carpeta /audio/
con los siguientes nombres:
- final-act-background-1.m4a
- final-act-background-2.m4a
- ending-background.mp3
- boom-sound-effect.mp3
- wave-sound-effect.mp3
